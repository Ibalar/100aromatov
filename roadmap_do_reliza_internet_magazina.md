# 🚀 Roadmap до релиза интернет-магазина

**Стек:** Laravel 12 + MoonShine 4\
**Статус:** БД, модели, админка и базовая логика заказов реализованы

------------------------------------------------------------------------

# ЭТАП 1 --- Завершение Backend Core (Критично для запуска)

## 1.1 Checkout (Frontend + Backend)

-   Форма оформления заказа
-   Валидация данных
-   Интеграция с OrderService
-   Обработка ошибок
-   Telegram-уведомление

## 1.2 Observer для PriceLog

-   Автоматическая запись при изменении цены
-   Логирование старой и новой цены
-   Привязка к пользователю (если админ меняет цену)

## 1.3 Enum-статусы заказа

-   new
-   paid
-   processing
-   shipped
-   completed
-   canceled

## 1.4 Email-уведомления

-   Клиенту
-   Администратору
